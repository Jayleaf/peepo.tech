COMMANDS

- AFK Set an AFK status.Arguments: <reason>
- Changelog Read the current changelog for the bot.Arguments: None
- Command Enable or disable a command. ⚠️ ADMIN ONLY.Arguments: <enable/disable> <command>
- Gayify Gayify someone's profile picture.Arguments: <member>
- Gayometer Check how gay you or someone else is!Arguments: optional: (<member>)
- Help Leads you to this page.Arguments: None
- Reddit Get a random post from a specified subreddit.Arguments: <subreddit>
- Remind Reminds you at a certain time about a specified reason.⚠️ Requires you have DMs on.Arguments: <set/delete/list> <days/hours/minutes/seconds (e.g. p!remind set 3d 5h 30m 5s)>
- Required Role Specifies what role a command requires.⚠️ ADMIN ONLY.Arguments: <command> <rolename/roleid>
- Slap Slap Someone!Arguments: <member>
- Stupid Fuck Call someone a stupid fuck, but in image form.Arguments: <member>
